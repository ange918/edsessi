# Maison de Haute Couture - Site Web

## Vue d'ensemble
Site web élégant et haut de gamme pour une Maison de Haute Couture béninoise inspirée du modèle d'Edi Sessi.
Le site reflète : luxe, élégance, modernité, héritage culturel et savoir-faire artisanal.

## Architecture du projet
- **Technologies** : HTML5, CSS3, JavaScript vanilla
- **Style** : Palette noir/blanc/gris (inspiration Chanel/Dior)
- **Design** : Luxe minimaliste, animations douces, responsive

## Structure des pages
1. **index.html** - Accueil avec hero fullscreen
2. **about.html** - À propos avec histoire, FAQ, blog, statistiques
3. **collections.html** - Grille des collections + sous-pages
4. **surmesure.html** - Sur-mesure avec processus et formulaire
5. **boutique.html** - Prêt-à-porter avec filtres
6. **galerie.html** - Lookbook avec lightbox et Instagram
7. **presse.html** - Actualités et presse
8. **contact.html** - Contact avec maps et formulaire

## Fonctionnalités JavaScript
- Menu sticky avec effet transparent au scroll
- Accordéon FAQ animé
- Compteurs statistiques animés
- Lightbox pour galerie
- Filtres produits
- Menu mobile responsive
- Animations fade-in, parallax, slow reveal
